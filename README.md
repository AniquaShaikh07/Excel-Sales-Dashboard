
# Sales Dashboard (Excel)

## Overview
This project is an Excel-based sales dashboard designed to analyze sales performance, revenue trends, and product insights.

## Goal of the Dashboard
To provide clear visibility into sales metrics and support data-driven business decisions.

## Data Source
The dataset consists of structured sales data including order details, products, customers, regions, revenue, and profit.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & KPIs
